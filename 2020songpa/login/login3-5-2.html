서울을 이끄는 송파구 통합로그인

송파구청 홈페이지 서비스를 이용함에 있어 안전한 인터넷 사용을 위하여
휴대전화와 민간아이핀 인증으로 본인확인을 실시하고 있습니다.

  • 통합회원 로그인
  • 비회원 로그인
  • 선택됨회원가입
  1. 약관동의
  2. 본인확인
  3. 현재단계정보입력
  4. 가입완료
정보 입력

  • 로그인 정보 입력

    주의사항 필수표시는 필수 항목 입니다.
    로그인 정보 입력 - 아이디, 비밀번호, 비밀번호 확인
    주의사항6 ~ 12자의 영문소문자, 숫자, _ , - 만 입력 가능합니다. 아이디 시작은 영문소문자만 가능
    주의사항8 ~ 12자의 영문소문자+숫자+특수문자(! @ # \ $ % ^ / * - +) 필수 입력입니다.

  • 법정대리인 정보 입력 (어린이 회원(14세 미만)은 보호자의 동의 및 인증 절차 후 가입이 가능합니다.)

    주의사항 필수표시는 필수 항목 입니다.
    법정대리인 정보 입력 - 법정대리인명, 법정대리인 연락처

  • 개인정보 입력

    주의사항 필수표시는 필수 항목 입니다.
    개인정보 입력 - 이름, 생년월일, 성별, 주소, 핸드폰, 전화번호, 이메일, 송파소식 메일 수신설정, 구정소식 SNS 수신설정
    생년월일 필수
    성별
    주소 필수
    송파소식 메일 수신설정 주의사항공연정보, 새소식, 생활편의정보, 교육강좌정보 등 유용한 정보메일발송
    구정소식 SMS 수신설정 주의사항민원처리상태, 접수 및 예약처리결과 등 신속한 정보 발송
